# 🏢 RentVerse

> **Where Renting meets Another Dimension**  
> A centralized digital solution for rent tracking, maintenance, and seamless tenant management.

## 📖 About the Project
Traditional property management often involves messy spreadsheets, scattered WhatsApp messages, and lost physical receipts. **RentVerse** is a role-based web application built to solve this problem for independent landlords. It provides a unified digital ecosystem where owners can manage their properties, tenants can pay rent and report issues, and technicians can track their repair tasks—all in one secure, easy-to-use platform.

## ✨ Key Features
The system is divided into four secure, role-based dashboards:

* **🏠 For Owners:**
  * Add new properties and upload property images.
  * Assign registered tenants to specific properties.
  * Review, approve, or reject uploaded rent payment receipts.
* **👤 For Tenants:**
  * View assigned lease details and monthly rent.
  * Upload screenshots of rent payment receipts for approval.
  * Raise maintenance tickets for property issues (e.g., broken plumbing).
* **🛠️ For Technicians:**
  * View a dedicated to-do list of pending maintenance requests.
  * Update task statuses to "Completed" once fixed.
* **📊 For System Agents (Admins):**
  * View platform-wide statistics (total users, properties, active leases, and revenue).

## 💻 Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **UI Framework:** Bootstrap 5, Bootstrap Icons
* **Backend:** Core PHP
* **Database:** MySQL
* **Architecture:** Client-Server with Single Page Application (SPA) style navigation.
